### Playwright 自动化测试

本次测试使用 OpenClaw Browser Control Server（底层使用 Playwright）进行：

1. **启动浏览器**
   ```
   browser action=start profile=openclaw
   ```

2. **打开页面**
   ```
   browser action=open url=http://localhost:5173/
   ```

3. **截图验证**
   ```
   browser action=screenshot
   ```

4. **快照分析**
   ```
   browser action=snapshot refs=role
   ```

5. **执行点击**
   ```
   browser action=act kind=click ref=e42
   ```

6. **页面刷新**
   ```
   browser action=act kind=press key=F5
   ```
