# 🎮 有趣记单词 (Word Fun)

> 一个帮助小朋友趣味学英语的 React 应用

![React](https://img.shields.io/badge/React-18-blue)
![Vite](https://img.shields.io/badge/Vite-5-purple)
![Tailwind](https://img.shields.io/badge/Tailwind-3-cyan)

## ✨ 功能特色

### 🎯 三种游戏模式

| 游戏 | 描述 | 技能 |
|------|------|------|
| 单词消消乐 | 配对单词和图片 | 词汇识别 |
| 拼写挑战 | 重新排列字母拼写单词 | 拼写能力 |
| 听音选词 | 听发音选择正确单词 | 听力理解 |

### 📚 词库主题

- 🐾 动物 (Animals) - 10个单词
- 🍎 水果 (Fruits) - 10个单词  
- 🎨 颜色 (Colors) - 10个单词

### 🏆 进度系统

- 💰 金币奖励
- ⭐ 星星收集
- 📊 本地存储进度

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 🛠️ 技术栈

- **前端框架**: React 18
- **构建工具**: Vite 5
- **样式**: Tailwind CSS 3
- **语音**: Web Speech API
- **存储**: localStorage
